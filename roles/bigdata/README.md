# Components

- NIFI : Data Flow
- Zookeeper : Config Mgmt
- HBase : Storage (file)
- Solr : Data Search
- Atlas : Data Governance

Optional:
- Hadoop : Storage / Map Reduce
- Kafka : Messaging pub/sub data streams
